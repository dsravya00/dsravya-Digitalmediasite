Exercise 6

In Exercise #6, I chose a Christmas theme and created three different page states. You can explore each one by pressing the keyboard keys 1, 2, 3 or 4. For the design, I incorporated the Christmas theme along with elements from Exercise #6. The basic structure includes a tree and various background elements. There are also three different sound variations: the first page plays a piano version of "Merry Christmas," the second page features another Christmas song, and the third page includes Santa Claus’ "Ho-ho-ho. and fourth page is plays a piano version of "Merry Christmas". Additionally, I’ve added jingle bell sounds for the buttons.

Final Project 4

In Final Project 4, I have developed a comprehensive setup centered around a Christmas theme. This includes four pages with different states, featuring music, buttons, background elements, and various effects incorporated throughout. To enhance the concept, I’ve provided descriptions of ornaments that symbolize few object decorated on the Christmas tree, with new items being added to the tree as the project progresses. One of the challenges I faced was implementing the wireframe requirements. I included a drag-and-drop feature to decorate the tree and added code for rotating the tree, but it doesn’t seem to work properly on scene page 2. Despite this, I have followed the Project 4 requirements checklist and made sure to cover all aspects in my final project.
